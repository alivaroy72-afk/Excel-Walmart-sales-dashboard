# Walmart Sales Performance Dashboard

## Project Overview

This project is an interactive Excel dashboard created to analyze Walmart sales data and identify business trends. The dashboard provides insights into sales performance, customer behavior, delivery efficiency, and order distribution. It was built as part of my data analytics learning journey to strengthen my Excel, Power Query, and dashboard development skills.

---

## Objective

The objective of this project is to transform raw sales data into meaningful business insights using Microsoft Excel. The dashboard helps answer questions such as:

- How do sales change over different months?
- Which customers generate the highest sales?
- Which states contribute the most revenue?
- What is the distribution of order values?
- How many orders are delivered on time?
- Which order type is most common?

## Data Cleaning

Before building the dashboard, the dataset was cleaned using Power Query.

The cleaning process included:

- Removing duplicate records
- Handling null values and blank entries
- Correcting incorrect data types
- Cleaning price columns
- Formatting rating and sales-related columns
- Removing unnecessary records
- Preparing the dataset for analysis

---

## Dashboard Features

The dashboard includes:

- Monthly Orders vs Sales Trend
- Top 10 Customers by Sales
- Top 5 States by Sales
- Sales by Product Category
- Delivery Progress (On-Time vs Delayed)
- Order Type Distribution
- Order Value Distribution
- Interactive Timeline Filter
- City Slicer

---

## Key Business Insights

- California generated the highest sales among all states.
- Most orders belonged to the Multi order category.
- Around 93% of deliveries were completed on time.
- Low-value orders accounted for the majority of transactions.
- Sales showed noticeable growth towards the end of the year.
- A small group of customers contributed significantly to total sales.
